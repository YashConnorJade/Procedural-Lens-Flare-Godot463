# Procedural-Lens-Flare-Godot-4.6.3
A procedural lens flare effect inside Godot 4.6.3. 
It uses a shader that creates halo effects, which mimics a real life lens flare effect of a camera.

You should see the scene from -Z axis. I have implemented a fade system where the lens flare is only visible from the front.
And Front in Godot is -Z. So the Lens flare Halos will only be shown if the Sprite3D is seen from -Z axis view.
